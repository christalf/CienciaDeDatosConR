1 Primer ejercicio - Housing

The American Community Survey distributes downloadable data about United States communities. Download the 2006 microdata survey about housing for the state of Idaho using download.file() from here:

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2Fss06hid.csv

and load the data into R.

The code book, describing the variable names is here:

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FPUMSDataDict06.pdf

and it’s already downloaded and available here:

Apply strsplit() to split all the names of the data frame on the characters “wgtp”.

What is the value of the 123 element of the resulting list?

  • The correct answer is: “” “15”

2 Segundo ejercicio - Gross Domestic Product

Load the Gross Domestic Product data for the 190 ranked countries in this data set:

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv

Remove the commas from the GDP numbers in millions of dollars and average them. What is the average?

  • The correct answer is: 377652.4

Original data sources:

  • http://data.worldbank.org/data-catalog/GDP-ranking-table

2.1 Solución

## Get the Gross Domestic Product data for the 190 ranked countries.
## The downloading was done and the file stored as "gdp-rank.csv" in "3_data" folder.
# if(!file.exists("./3_data")) {dir.create("./3_data")}
# fileUrl2 <- "https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv"
# download.file(fileUrl2, destfile = "./3_data/gdp-rank.csv", method = "curl")

## Fa schiffo sto file:
## - The column names are incomplete and occupie two lines, the 3rd and 4th
## - The 5th line is blank.
## - There are 190 ranked countries, then the file contiues with not available data, and
##   we only want to read the 190 ranked countries (from line 5 to line 195, that is,
##   we want to read 190 lines).
gdp_rank <- read.csv("3_data/gdp-rank.csv", skip = 4, nrows = 190)
sapply(gdp_rank, class)
        X       X.1       X.2       X.3       X.4       X.5       X.6       X.7 
 "factor" "integer" "logical"  "factor"  "factor"  "factor" "logical" "logical" 
      X.8       X.9 
"logical" "logical" 

gdp_rank <- gdp_rank[, c(1, 2, 4, 5)]
names(gdp_rank)
[1] "X"   "X.1" "X.3" "X.4"

colnames(gdp_rank) <- c("CountryCode", "Rank", "Country", "Ammount")
sapply(gdp_rank, class)
CountryCode        Rank     Country     Ammount 
   "factor"   "integer"    "factor"    "factor" 

## Remove the commas from the GDP numbers in millions of dollars and average them.
## What is the average?
gdps <- gsub(",", "", gdp_rank$Ammount)
gdps <- as.integer(gdps)
mean(gdps)
[1] 377652.4

4 Cuarto ejercicio - Gross Domestic Product

Load the Gross Domestic Product data for the 190 ranked countries in this data set:

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FGDP.csv

Load the educational data from this data set:

https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FEDSTATS_Country.csv

Match the data based on the country shortcode. Of the countries for which the end of the fiscal year is available, how many end in June?

  • The correct answer is: 13

Original data sources:

  • http://data.worldbank.org/data-catalog/GDP-ranking-table

sampleTimes = index(amzn)
